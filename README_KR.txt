makeOne.cpp

makeOne04C03 함수가 최종형이다.
0<N<30001 의 정수 N을 /5, /3, /2, -1 4가지 방법을 이용하여 1로 만드는 최소 계산횟수를 구하는 것이다.
기존의 4가지 경우를 모두 따져서 경로를 찾아가는 방법이 있다.
그에 비해 이 함수는 모든 경우의 수를 살펴볼수 있으면서, 알아보기 더 쉬운 자료형으로 표현하였다.
5, 3, 2 의 최소 공배수는 30이다.
현재의 숫자가 어느 숫자로 나누어지는 수인지 30의 주기로 반복되어 나타난다는 것이다.
table 2차원 배열이 그것을 나타내는데, 각 원소 배열이 나타내는 것은
배열 순서가 주기를 의미하고, 배열 내부는 자기가 계산 가능한 시점이면 0, 그렇지 않다면 1 으로 하였고, 순서대로 /5, /3, /2, -1 이다.
B_tree 2차원 배열에 계산 과정이 저장되는데, 일종의 이진트리 모습으로 만들었다.
2차원 배열 첫번째 원소에 입력받은 N 을 넣고, B_tree 를 순서대로 읽으면서
숫자를 4가지 방법으로 계산한 값을 또다른 1차원 배열 idx 에 매번 저장하고
숫자를 30으로 나눈 나머지 값이 table 원소 배열을 고르게 되고, 내부에서 0 이 되는 i 값을 찾는다.
idx[i] 값이 B_tree 의 다음 원소 배열에 삽입되고, 그 과정을 1이 될때까지 반복하게 된다.
